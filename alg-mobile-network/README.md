# ALG Mobile Network Platform

A responsive, enterprise-grade multi-brand website platform engineered for the ALG Mobile Network. The platform hosts a centralized hub that dynamically bridges to dedicated, isolated landing experiences for three independent mobile brands: Fone, GAP, and Flipper.

## 🛠️ Tech Stack & Focus Areas
* **Frontend Core:** Semantic HTML5 / Modular CSS3 / Modern Vanilla JavaScript
* **Design Methodology:** Mobile-First Responsive Design, Cross-Browser Compatibility, Consistent UX/UI Design Language

## 📈 Core Business Value & Engineering Impact
* **Multi-Brand Scaling:** Demonstrates how a single parent entity can manage multiple distinct product design schemas (Fone, GAP, Flipper) without creating separate, disjointed codebases.
* **Responsive Fluidity:** Engineered using CSS Flexbox/Grid layouts to ensure seamless user engagement across smartphones, tablets, and desktop displays.
* **Performance-Focused UI:** Utilizes optimized assets and decoupled vanilla JavaScript to minimize asset weight, leading to fast loading speeds and high SEO discoverability scores.

## 📁 Platform Architecture Breakdown
The project is architected into specific user flows:
1. **Central Portal:** The core ALG Network hub directing target consumer demographics to the appropriate brand vector.
2. **Fone Landing Experience:** Tailored UI elements highlighting premium data features and network capabilities.
3. **GAP Landing Experience:** Focused on budget-conscious or high-value multi-line structures.
4. **Flipper Landing Experience:** A distinct design layout styled for youth or dynamic data-tier audiences.
